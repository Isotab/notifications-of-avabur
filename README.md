# notifications-of-avabur
Notifies users (with sounds and toasts) of events within Relics of Avabur.

## Supported events
*`Fatigued`
*`Event countdown started`
*`Event started`
*`Harvestron available`
*`Construction available`
*`Whispers`
*`Quest Complete`
